## 简介

缠中说禅（本名李彪），中国股市技术分析大师，缠论创立者。2006-2008年新浪博客活跃，以“本ID”自称。其理论体系包括笔、线段、中枢、背驰、买卖点、走势分解、分型理论、级别嵌套等。主张顺势而为，反对预测，强调纪律和系统性。
